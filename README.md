# AirGuard AI

AirGuard AI is an AI-powered urban air quality intelligence platform designed to help smart cities predict pollution before it becomes dangerous.

## Features
- Real-time AQI monitoring
- AQI prediction (24–72 hours)
- Pollution hotspot detection
- Smart citizen alerts
- Government recommendations

## Tech Stack
- Python
- Flask
- TensorFlow
- Scikit-learn
- MongoDB
- APIs (Weather, AQI, Traffic)
